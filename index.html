জাভাস্কিপ্ট একটি সিঙ্গেল থ্রেডেড PROGRAMING লেংগুয়েজ। আমরা যখন জাভাস্কিপ্ট কোড লিখি সেটা browser এর কাছে যাওযার পর browser সেটা V8 ইন্জিন এর কাছে দিয়ে দেয়। V8 সেটাকে Interpretation ও Compilation এর মাধ্যমে machin code এ পরিবর্তন করে। interpretation বার বার ফাইল কোড নিয়ে এসে কাজ করে। এবং comoailer আমাদের কোড গুলো এক বারে মেশিন কোডে পরিবর্তন করে এবং এক সাথে কোড শেষ পর্যন্ত চলতে থাকে।

Browser এ থাকা v8 Engine আমাদের জাভাস্কিপ্ট কোড রান করে। কিন্তু জাভাস্ক্রিপ্ট ইন্জিন কখনোই একসাথে একটার বেশি স্টেটমেন্ট প্রসেস করে না। Engine এ থাকা HEAP Call Stack এর মধ্যে Heap মেমোরি মেনজমেন্টের কাজ করে। callstack আমাদের কল করা Function কে একটা একটা করে কল করে দেয় কিন্তু একই সাথে Asynchronous function কে কল করতে পারে না তখন কল স্টেক সেই ফাংশনকে WEB API এ পাঠিয়ে দেয়। Call stack থাকা Synchronous Function কল করা শেষ web API সেটা Call back QUEUE এ পাঠিয়ে দেয় এবং QUEUE থেকে একটা একটা Function কে Event loop আবার Call stack এ পাঠিয়ে দেয়। সেটাই হল জাভাস্ক্রিপ্ট ইভেন্ট লুপ।
